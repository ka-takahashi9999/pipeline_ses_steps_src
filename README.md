  # pipeline_ses_steps_src

  SESマッチングシステム用パイプラインのソースコード管理リポジトリです。

  ## このリポジトリに含めるもの
  - `common/`
  - 各 step 配下の `00_tool/`
  - 実装・運用に必要な最小限の設定ファイル
  - `AGENTS.md`

  ## このリポジトリに含めないもの
  - `01_result/`
  - `02_confirm/`
  - `99_execution_time/`
  - `json` / `jsonl` などの生成物
  - 一時確認ファイル
  - 実行ログなどの成果物

  ## 目的
  本番・作業用ディレクトリとは分離して、GitHub 上ではソースコードのみを管理するためのリポジトリです。

  ## 運用方針
  - 日常作業は元の作業ディレクトリで行う
  - GitHub に反映したいコードのみ、このリポジトリへ同期する
  - 生成物や中間成果物はコミットしない

  ## 注意
  各 step の入出力 JSONL スキーマは、実データ側の運用と整合を保った上で変更すること。
